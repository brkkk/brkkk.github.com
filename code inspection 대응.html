FindBugs
  • FindBugs requires JRE (or JDK) 1.7.0 or later to run. However, it can analyze programs compiled for any version of Java, from 1.0 to 1.8.
  • 컴파일된 바이트 코드에서 동작
설치 및 실행 방법
  1. 이클립스 실행
  2. Help > Eclipse MarketPlace > findbugs 검색
  3. Preferences > FindBugs 에서 옵션 설정.

Checkstyle
  • Checkstyle is a development tool to help programmers write Java code that adheres to a coding standard.
  • 자바코딩스타일 체크
설치 및 실행 방법
  1. 다운로드 페이지 접속
  2. 설치파일 다운로드 후 압축 해제 (features, plugins)
  3. eclipse/features 에 파일 복사
  4. eclipse/plugins 에 파일 복사
  5. Preferences > Checkstyle 옵션 설정.
  6. 프로젝트 선택 > Properties > Checkstyle 선택
  7. Main 탭에서 Checkstyle active for this project 체크
  8. Local Check Configurations 선택 > New 선택 후 사용자 설정파일을 만든다.
  9. Main 탭으로 이동 > 사용자 설정파일 선택 > Configurations 선택
  10. 모듈중에 사용할 모듈을 추가 후 OK 선택
  11. 프로젝트 선택 > 마우스 오른쪽 단추 선택 > Checkstyle > Check Code Checkstyle 선택
  12. Checkstyle violations 에서 목록 확인.

일반적으로 사용자 설정파일을 추가하지 않을 경우 Sun Checks(Eclipse) Global 선택.

효과
  • 체크스타일 모듈 선택 창에서 해당 모듈의 정규식(format)을 수정할 수 있음.
  • Naming Conventions, Coding Problems 등 체크 가능


PMD
  • PMD is a source code analyzer
  • 소스코드에서 동작
  • 전자정부 표준프레임워크 개발도구에 PMD Plugin으로 포함되어 배포되고 있다.
설치 및 실행 방법
  1. Eclipse 실행
  2. Help > Install New Software…
  3. Install Path : http://sourceforge.net/projects/pmd/files/pmd-eclipse/update-site/
  4. Preferences > PMD 옵션설정.
  5. Preferences > PMD > Rule Configuration 선택
  6. 사용할 룰 다중선택 후 .ruleset 파일 EXPORT
  7. 프로젝트 선택 > 마우스오른쪽 단추 선택 > Properties 선택
  8. Enable PMD 체크
  9. Use the ruleset configured in a project file 에 EXPORT한 ruleset 파일 지정 후 OK
  10. PMD > Violations Overview 에서 목록 확인.
  11. Violations Outline 에서 상세 확인 가능.

Eclipse Code Formatter Clean Up 기능

설치 및 실행 방법
  1. Eclipse 실행
  2. Preferences > Java > Code Style > Clean Up
  3. New… 선택
  4. Profile name 입력 후 edit 창에서 수행할 항목 체크
  5. OK 선택 후 Clean Up 수행
효과
  • Remove unused imports
  • Organize imports
  • Use blocks in if/while/for/do statements
  • 전역 private 변수에 대해 method 내에서 사용할 경우 this 명시
  • serial version ID 추가


JAutodoc

JAVA 파일 자동 주석 달기
ALT + SHIFT + J 로 주석가능하나 해당 메소드 라인에서 해야 함.
한번에 메소드 주석을 일괄적으로 추가해 주는 것 필요했음.

설치가 완료되면 Clean Up 에서도 사용 가능한데 파일 헤더 주석 넣는 부분만 있음.

설치 및 실행 방법
  1. Eclipse 실행
  2. Help > Install New Software…
  3. Install Path : http://jautodoc.sourceforge.net/update/
  4. Preferences > Java > JAutodoc 에서 옵션 값 설정 가능. JAutodoc > Templates 에서 주석 template 수정 가능.
  5. Project Explorer 선택
    (Project Explorer 내의 가장 하위 패키지 단위에서 실행 가능)
  6. 파일 선택 후 오른쪽 단추 선택
  7. JAutodoc > Add Javadoc for Members.. 선택
